# 找牛游戏（Find the Cow）

参考抖音小游戏的找牛玩法复现的益智解谜游戏，结合**约束满足**与**地图生成**，类似 Star Battle / Parks 变体。

## 规则

- **棋盘**：n×n 网格（默认 8×8），由 n 种颜色的联通区域组成。
- **小牛**：每种颜色有且仅有 1 头小牛；每行、每列均有且仅有 1 头小牛；小牛在 8 邻域内不能相邻。
- **操作**：点击格子循环切换「标为牛 🐄」→「排除 ×」→「空白」。

## 生成算法

1. **放置牛（种子）**：在 n×n 上随机放置 n 个格子，满足每行每列唯一且 8 邻域不相邻（随机尝试 + 有限次重试）。
2. **生成区域**：以这 n 个格子为种子，多源 BFS 随机扩张，直到填满网格，保证每种颜色区域联通且内含唯一合法牛位。

## 运行

用浏览器直接打开 `index.html` 即可，无需服务器。

```bash
# 可选：用本地服务器避免 file:// 限制
cd FindTheCow
python -m http.server 8080
# 打开 http://localhost:8080
```

## 文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | 页面结构、规则说明与控件 |
| `style.css` | 棋盘与格子样式、区域颜色、牛/排除/错误动画 |
| `game.js` | 种子生成、多源 BFS 区域、点击逻辑、违规提示、通关判定 |
